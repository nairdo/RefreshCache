Want to speed up your website, but don’t know how? There are some tried and true techniques for improving website performance (etagging, caching, minifying, etc...) but it’s hard to keep track of all of the different files and settings that you change along the way. We’ve made the process much easier by building an Arena module that modifies and monitors these changes for you. With this management console, you’ll have your site humming along in no time.
